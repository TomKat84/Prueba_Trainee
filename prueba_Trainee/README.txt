Para la ejecución de este proyecto primero se va a necesitar ciertos requerimientos:
1. Python versión 3.9 o mayor
2. un editor de texto o un IDE
3. PC Windows (recomendable)

los pasos para ejecutar este proyecto va a variar entre los usuarios, pero yo les mostrare los pasos que yo use:
1. abrir el botón de inicio de Windows y escribir "idle" y les saldrá una aplicación de Python.

2. darle click a "File" y selecciona "Open". Selecciona donde esta guardado el archivo de Python que se desea abrir y se les abrirá otra ventana de Python con los códigos.

3. darle click a "Run" y selecciona "Run Module"
